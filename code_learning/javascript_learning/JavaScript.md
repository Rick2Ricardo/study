# JavaScript 基础

## JavaScript 介绍

### 1.1Javascript是什么

1、Javascript(是什么？)

是一种运行在客户端（浏览器）的编程语言，实现人机交互效果。

2、作用（做什么？）

- 网页特效
- 表单验证
- 数据交互
- 服务端编程（node.js）

3、Javascript的组成（有什么？）

ECMAScript：

规定了js基础语法核心知识 如：变量、分支语句、循环语句、对象等等

Web APIS:

Dom 操作文档，比如对页面元素进行移动、大小、添加删除等操作

BOM 操作浏览器，比如页面弹窗，检测窗口宽度、存储数据到浏览器等等
